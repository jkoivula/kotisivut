Website

05.2016

This website was created as our Web development course final project. The website is for a make-believe company called Moon Travels that offers trips to different locations around the Solar System.

We build the website design using web techniques such as HTML5, CSS3 and Bootstrap. We also added some functionality to the navigation bar by using jQuery. Using PHP we created the websites multiple language options, functioning 'Make an offer'-form that sends the offers to our MySQL database and an administrator view for managing made offers.

Moon Travels website was created by a group of three students including Pasi Pahkuri and Joni Koivula.

Corona SDK Mobile Game

06.2016

This project was the main focus of a Mobile Game Programming Fundamentals summer course. During the summer course we needed to create a simple mobile game of any genre, using Corona SDK framework. The mobile game was programmed completely with Lua.

My game is an endless runner. Goal of the game is to platform the character across the level successfully by controlling the jumping function. Game levels are randomly generated, making the experience different everytime. The game randomly generates how many platforms are on the level, how wide and high they are (this is done for each individual platform) and how far apart from each other they are.
I created the graphical assets using Inkscape.

I created the mobile game by myself.

Techniques:

  • Lua
  • Corona SDK

Node.js & Socket.io Web Application

02.2017~

This web application is the current result of the curiosity of my friend and I to familiarize with Node.js and Socket.io. Also, we both wanted to sharpen our skills in JavaScript.

The application is still very much a work in progress idea wise but we've playing around with the functionalities Socket.io offers to make a simple chat application.

Users can enter a chatroom and type messages which are displayed in a HTML5 canvas using P5 library. The application keeps track on current users and color statistics. In this application users can choose a messaging color and the color with the most messages determines the color theme for the application.

This application is being developed by Mindi Laihonen and Joni Koivula.

Techniques:

  • JavaScript
  • Socket.io
  • Node.js
  • P5
